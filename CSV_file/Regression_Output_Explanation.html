Regression Output Explanation

This project uses the official World Bank Worldwide Governance Indicators rule-of-law series RL.EST (Rule of Law: Estimate) as the rule-of-law variable in the regression outputs.

Interpretation

  • Higher RL.EST values indicate stronger perceived rule of law.
  • Lower RL.EST values indicate weaker perceived rule of law.
  • The measure is a standardized governance estimate, not a funding or investment index.
  • The values should not be interpreted as percentages.

Regression Reading

The regression figures in this repository should be interpreted as relationships between GDP growth, GDP level, life expectancy, and RL.EST.
